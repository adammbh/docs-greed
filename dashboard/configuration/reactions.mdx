## Button Roles
Set up role assignment buttons for messages. (Coming soon: Button roles will be available in the dashboard in a future update.)

## Anti-Self-React
Prevent users from reacting to their own messages.
- **Enable Anti-Self-React**: Toggle to prevent users from reacting to their own messages.
- **Ignored Users**: Users who can react to their own messages.
- **Ignored Roles**: Roles that can react to their own messages.

<Note>
You can add ignored users or roles using the respective buttons.
</Note>
